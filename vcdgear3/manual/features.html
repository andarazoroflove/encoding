Features in the GUI version include:

  • Graphical user interface

  • Multiple language support

  • Faster processing performance

  • Batch processing support

  • RAR / unRAR support (with batch processing)

  • VCD and SVCD image generation (yes, batch supported here too!)

  • an Easter egg  :-)

The console (command line) version only has the main extraction/conversion routines minus all the
eye candy found in the GUI.
