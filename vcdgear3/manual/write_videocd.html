Writing VideoCD Images to CD

 

To write a (Super) VideoCD image to CD, you will need to use an external writing program such as Nero (www.ahead.de), CDRWin (www.goldenhawk.com), or FireBurner (www.fireburner.com).

Two kinds of (Super) VideoCD images can be generated with VCDGear: .cue/.bin and .toc/.bin.  .toc/.bin can be written using certain programs on the Linux platform.  .cue/.bin can be written using Nero/CDRWin/FireBurner.

These images must be written in this format:  MODE 2 XA / 2352-bytes per sector (Form 2).

With most CD writing software, you will not need to worry about setting these parameters as the index file will automatically tell the software what format your image is in.

Follow your CD writing software's manual on how to burn a disc image.  Then pop it into your VCD/DVD player and enjoy!
